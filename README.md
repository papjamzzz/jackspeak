# Jackspeak

**Cat vocalization interpreter.**

Record what your cat says. Claude classifies it. Build a behavioral baseline over time.

---

## What It Does

Records short audio clips of your cat and sends them to Claude Haiku for classification. Over time, builds a per-cat vocabulary — what does this specific cat's meow actually mean?

---

## Features

- Live audio recording in browser
- Claude Haiku classification with confidence score
- Per-cat behavioral baseline that builds over time
- Dark-theme machine UI

---
